# 🐍 Python E-Commerce Data Analysis

## 📌 Project Overview

This project was completed as the Final Project for the MySkill Data Analyst Bootcamp.

The project focuses on analyzing an e-commerce transaction dataset using Python in Google Colab. The analysis includes data preparation, exploratory data analysis (EDA), data visualization, and answering business questions to generate actionable business insights.

---

## 🎯 Objectives

- Clean and prepare raw transaction data.
- Explore customer purchasing behavior.
- Perform business analysis using Python.
- Generate insights to support business decision-making.
- Present recommendations based on analytical findings.

---

## 🛠 Tools & Libraries

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Project Workflow

### 1. Data Preparation

The dataset was prepared before analysis by performing several preprocessing steps:

- Checking duplicate records
- Handling missing values
- Converting data types
- Data cleaning
- Preparing the dataset for analysis

---

### 2. Exploratory Data Analysis (EDA)

Exploratory analysis was conducted to understand:

- Customer transactions
- Product performance
- Sales trends
- Customer behavior

---

## 📊 Business Questions

### Question 1

Analyze customer transaction behavior to identify purchasing trends and business opportunities.

**Output**

- Customer transaction analysis
- Summary statistics
- Business insights

---

### Question 2

Analyze product category performance to determine the best-performing categories and identify improvement opportunities.

**Output**

- Category comparison
- Sales analysis
- Visualization
- Recommendations

---

### Question 3

Identify customers who completed checkout but did not complete payment during 2022 for remarketing purposes.

**Output**

- Customer ID
- Registered Date
- Marketing target list

---

### Question 4

Analyze customer transaction patterns to support business strategy and customer segmentation.

**Output**

- Customer analysis
- Visualizations
- Business recommendations

---

## 📈 Key Insights

Some important findings from this project include:

- The dataset required preprocessing before analysis.
- Customer purchasing behavior varies across different transaction periods.
- Several product categories generated higher sales than others.
- There are customers eligible for remarketing campaigns due to incomplete payments.
- Data-driven insights can support better marketing and business decisions.
